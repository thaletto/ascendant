# Ascendant

Ascendant is a Python project designed to provide functionalities related to astrological calculations and interpretations.

## Core Functionalities:

- **Chart Calculations**: This module handles the computation and analysis of astrological charts.
- **Dasha System**: Implements the Dasha system for planetary periods, crucial for predictive astrology.
- **Yoga Combinations**: Provides tools for identifying and interpreting various Yoga (planetary combinations) in a birth chart.
